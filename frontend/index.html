⚙️ Advanced controls

The Model options panel is meant for experienced CREST/EF5 users — it directly overrides the simulation window, time step, warm-up, model choice and calibration parameters that go into the control file.

If you'd rather not tune these by hand, the assistant in the chat box can find the event, set the time window, and pick sensible settings for you.
